Introduction

This is a WeiDU mod that restores the 'classic' movies from the original Baldur's Gate and select movies from Baldur's Gate II to Baldur's Gate: Enhanced Edition. The classic movies have been resized to the same dimensions as the new BG:EE movies (1280x720, 852x480, and 512x288) while maintaining their aspect ratios. It also re-enables the Beregost movie when you first travel to the area and restores a movie to the Elfsong Tavern that was cut from the original game. This mod was developed for Baldur's Gate Enhanced Edition v1.3+ and will be considered a beta version as long as BG:EE is being updated. Use at your own risk! Questions, comments, suggestions, and bug reports are of course welcome.

Installation

This mod is broken into two different GitHub branches: "BGEE-Classic-Movies:master" and "BGEE-Classic-Movies:LowResOnly". The first one (the "master" branch) contains everything needed to install and use the BG:EE Classic Movies mod. Under normal circumstances, this is the only part of the mod you will ever need. The second branch ("LowResOnly") contains the lower resolution versions of the movies (found in the "480" and "lo" folders in the movies folders in the game directory). I don't think these lower resolution versions are used on the Windows or Mac version of the game, but might be on other platforms (such as i-pad or Android tablet). If you want to install these along with the high resolution versions of the movies, extract the contents of this archive on top of the contents of the first one before installing the mod. Be sure to extract relative paths!

This is a WeiDU mod. To install, unzip the latest GitHub Release into the Baldur's Gate: Enhanced Edition game folder that has the chitin.key file (usually "Baldur's Gate Enhanced Edition\Data\00766") and double-click on setup-BGEEClassicMovies.exe. Follow the command prompts to install the desired components. To uninstall, double-click on setup-BGEEClassicMovies.exe again and select the Uninstall option.

If you have followed the above steps and the mod is still not installing properly, you might find reading The common Windows Vista and Windows 7 problems helpful. Particularly, sections 2.1, 2.2.1, and 2.2.2.

Frequently Asked Questions

Q: Why does MOVIDESC.2DA and Baldur.ini (or Baldur.lua) need to be modified?
A: The Baldur.ini (or Baldur.lua) and MOVIDESC.2DA are modified so that the movies can be viewed in the "Movies" dialog of the GUI. Since not all of the movies from BG1 are actually used in BG:EE, this is the only way to watch them.

Q: I hear strange noises when some movies are played.
A: BG:EE uses a crappy WEBM codec, so if you are having problems with the audio in the moves, this should be resolved in the next patch. The problem does not occur in the English movies, but in the Polish movies, for example, you can hear strange noises when played in BG:EE. BG2:EE used to have the same problem, but the newest patch updated the codec and now the engine plays them as expected.
